# Experiment 12: Second Largest Element

## Problem Statement

Given an unsorted array of size $N$, find the **2nd largest element** from the array without sorting the array.

## Input Format

* The first line contains a single integer $N$.
* The second line consists of $N$ integers of the array.

## Output Format

Print the second largest number in a new line.

### Example 1

**Input:**

```text
6
3 2 1 5 6 4
```

**Output:**

```text
5
```

### Example 2

**Input:**

```text
9
10 1 7 2 5 6 8 9 3
```

**Output:**

```text
9
```
